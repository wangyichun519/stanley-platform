## STEP 1：複製專案到你的電腦

把這個資料夾整個複製到你的電腦，然後：

```bash
# 進入專案
cd healthcare-platform

# 複製環境變數設定
cp .env.example .env
```

---

## STEP 2：設定環境變數

用任何文字編輯器打開 `.env`，填入：

```bash
# Gmail 設定（需開啟「應用程式密碼」功能）
EMAIL_USER=你的Gmail@gmail.com
EMAIL_PASS=你的應用程式密碼（16碼）

# 後台帳密（自己設）
ADMIN_USER=stanley
ADMIN_PASS=你的安全密碼

# LINE Pay（先填 sandbox 測試用）
LINE_PAY_CHANNEL_ID=（申請後填入）
LINE_PAY_CHANNEL_SECRET=（申請後填入）
LINE_PAY_MODE=sandbox

BASE_URL=http://localhost:3000
```

> Gmail 應用程式密碼設定：
> Google 帳號 → 安全性 → 兩步驟驗證 → 應用程式密碼

## STEP 5：Cowork 自動化任務

打開 **Claude Desktop → Cowork 分頁 → 指向 healthcare-platform 資料夾**

### 任務一：每日自動報表
```
/schedule

每天早上 08:00 執行：
讀取 platform.db 資料庫
統計：
1. 今日新增顧問預約
2. 今日課程報名人數
3. 今日商店訂單
4. 待處理事項

產生報表存到 reports/daily/YYYY-MM-DD.txt
```

### 任務二：開課前自動提醒
```
/schedule

每天早上 07:30 執行：
查詢 3 天後的課程
找出所有報名且已付款的學員 email
發送提醒信給每位學員
```

### 任務三：每週業績摘要
```
/schedule

每週一 09:00 執行：
統計上週各模組收入（顧問/課程/商店）
產生週報 reports/weekly/week-YYYY-Www.xlsx
```

## STEP 6：部署到 Vercel（上線）

```bash
# 安裝 Vercel CLI
npm install -g vercel

# 登入
vercel login

# 部署（照引導選擇）
vercel

# 設定環境變數到雲端
vercel env add EMAIL_USER
vercel env add EMAIL_PASS
vercel env add ADMIN_USER
vercel env add ADMIN_PASS
vercel env add LINE_PAY_CHANNEL_ID
vercel env add LINE_PAY_CHANNEL_SECRET
vercel env add LINE_PAY_MODE
vercel env add BASE_URL  # ← 填你的 Vercel 網址

# 正式部署
vercel --prod
```

完成後你的網站將在：
```
https://healthcare-platform-xxxxx.vercel.app
```

---

## STEP 7：LINE Pay 申請（Sandbox）

1. 前往：https://pay.line.me/tw/developers/techsupport/sandbox/testflow
2. 點選「申請測試帳號」
3. 取得 Channel ID 和 Channel Secret Key
4. 填入 `.env` 並重啟伺服器
5. 上線時改為正式帳號：LINE Pay 商家申請

## 🆘 常見問題

| 問題 | 解法 |
|------|------|
| `npm install` 失敗 | 確認 Node.js v18+ 已安裝 |
| 資料庫找不到 | 第一次啟動自動建立，正常 |
| LINE Pay 付款失敗 | 確認 Channel ID/Secret 正確，先用 sandbox 測試 |
| Email 發送失敗 | Gmail 需開啟「應用程式密碼」，不能用一般密碼 |
| Vercel 部署失敗 | 在 CLI 輸入 `claude "幫我修復 Vercel 部署錯誤：[錯誤訊息]"` |
| 後台無法進入 | 確認 `.env` 中 ADMIN_USER 和 ADMIN_PASS 已設定 |
